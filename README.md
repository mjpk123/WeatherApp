# WeatherApp

A simple and responsive Weather Application that allows users to check real-time weather conditions of any city using an external weather API.

🚀 Features
🌍 Search weather by city name
🌡️ Displays temperature, humidity, and weather condition
🌤️ Dynamic weather icons
📱 Responsive design (works on mobile & desktop)
⚡ Fast API-based data fetching
🛠️ Tech Stack
HTML
CSS
JavaScript
Weather API (like OpenWeather API)
📸 Preview

⚙️ How It Works
User enters city name
App sends request to Weather API
API returns weather data
Data is displayed on UI
🔑 API Setup
Go to OpenWeather website
Create a free account
Generate your API key
Replace API key in your JavaScript file:const apiKey = "YOUR_API_KEY";
